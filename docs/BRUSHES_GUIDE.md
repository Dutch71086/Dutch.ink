# HD Tattoo Brushes Guide

## Brush Properties Explained

### Size Range
- Minimum to maximum pixel size for the brush
- Larger sizes work better for big coverage, smaller for details

### Opacity Range
- Minimum to maximum opacity the brush can achieve
- Lower values create more transparent, layered effects

### Hardness (0-1)
- **1.0** = Sharp, hard edges (outlines, details)
- **0.5** = Medium softness (general shading)
- **0.0** = Completely soft (blending, smooth gradients)

### Spacing (0-1)
- Distance between brush stamps along a stroke
- **Lower** = More continuous, smooth strokes
- **Higher** = Dotted, scattered appearance

### Jitter (0-1)
- Random variation in brush placement
- **0.0** = No randomness (precise)
- **High** = Scattered, natural appearance

### Smoothing (0-1)
- Stroke smoothing and stabilization
- **High** = Very smooth, stabilized strokes
- **Low** = More responsive to hand movements

### Texture Depth (0-1)
- How pronounced the surface texture appears
- **0.0** = Flat, smooth
- **1.0** = Heavily textured

### Sensitivity Features
- **Pressure Sensitivity**: Responds to pen pressure for line weight
- **Tilt Sensitivity**: Responds to pen tilt angle
- **Velocity Sensitivity**: Responds to stroke speed
- **Color Dynamics**: Uses color palette for variation

## Usage Tips

### For Outlines
1. Start with **Outline Medium** (most versatile)
2. Use **Outline Thin** for fine details
3. Use **Outline Thick** for emphasis
4. Enable pressure sensitivity for natural variation
5. Maintain consistent opacity (0.9-1.0)

### For Shading
1. Start with **Shading Medium** (0.5-0.7 opacity)
2. Layer multiple passes for depth
3. Use **Shading Soft** for delicate shading
4. Use **Shading Hard** for bold shadows
5. Enable color dynamics for realistic tones

### For Fills
1. Choose size appropriate to area
2. **Solid Fill** for flat colors
3. **Textured Fill** for organic look
4. **Gradient Fill** for smooth transitions
5. **Stipple Fill** for artistic effect

### For Details
1. Zoom in to 200-400%
2. Use **Detail Fine** for small elements
3. Use **Eyeliner Detail** for eyes
4. Keep opacity high (0.9+)
5. Use zoom shortcuts for precision

### For Special Effects
1. **Watercolor** for artistic backgrounds
2. **Ink Splatter** for dynamic effects
3. **Glow Effect** for ethereal highlights
4. **Flame** for fire and energy
5. **Smoke** for atmospheric depth

### For Stippling
1. Build up dots gradually
2. Use **Stipple Fine** for delicate work
3. Use **Stipple Large** for bold effects
4. Layer multiple colors for depth
5. Use **Stipple Gradient** for tonal transitions

## Creating Custom Presets

Save your favorite brush configurations:

```python
brush_manager.create_preset(
    name="My Custom Shading",
    brush_id="shading_medium",
    size=12.0,
    opacity=0.6,
    hardness=0.55,
    spacing=0.18,
    smoothing=0.65,
    color=(50, 30, 20),  # Dark brown
    description="Custom preset for skin tones"
)
```

## Troubleshooting

### Brush too hard/soft
- Adjust hardness slider in properties panel
- Use different brush from category

### Brush not responding to pressure
- Verify pen/tablet is properly configured
- Check pressure sensitivity is enabled
- Try different pressure-sensitive brush

### Brush performance issues
- Reduce brush size
- Lower texture depth
- Use GPU acceleration settings
